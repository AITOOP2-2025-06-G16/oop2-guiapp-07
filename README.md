プログラムの実行方法  
ターミナルでプロジェクトディレクトリに移動します。  
cd oop2-2025-05-G03  
以下のコマンドでプログラムを実行します。  
python3 main.py  
※VS Code の「▶️」ボタンから実行すると、参照パスの問題で正しく動作しない場合があります。必ずターミナルから実行してください。  
プログラムを実行すると、ウィンドウに 「撮影」ボタン と 「合成実行」ボタン が表示されます。  
撮影ボタン を押すと、カメラ画像が取得されます。  
合成実行ボタン を押すと、カメラ画像とGoogleロゴが合成され、同時に画像が保存されます。  
保存後にウィンドウ下部に 「合成が完了し、保存しました！」 と表示されます。
