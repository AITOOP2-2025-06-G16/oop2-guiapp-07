プログラム実行方法　　  
「cd oop2-2025-05-G03」 に移動し、「python3 main.py」で実行する。  
vsコードの▶️で実行すると、参照がおかしくなるので、コマンドで実行。  
実行すると、撮影ボタンと、合成実行ボタンが出る。  
合成実行ボタンを押すと、同時に保存が行われ、最後に、「合成、保存が完了しました」という文が表示される。
